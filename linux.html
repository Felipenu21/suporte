Introdução ao Linux

Linux é um sistema operacional open source amplamente usado em servidores, desktops, dispositivos embarcados e ambientes de desenvolvimento.

Conhecer os comandos básicos, sintaxes e ferramentas do Linux é essencial para administradores de sistemas e profissionais de suporte técnico.

Ferramentas Importantes

  • htop: Visualização interativa dos processos, use htop.
  • netstat / ss: Verifica conexões e portas abertas. Ex: netstat -tulnp ou ss -tuln.
  • tcpdump: Captura pacotes da rede para análise. Ex: sudo tcpdump -i eth0.
  • lsof: Lista arquivos abertos por processos. Ex: lsof -i :80.
  • df / du: Monitoramento de espaço em disco.
  • cron / systemd timers: Agendamento de tarefas automáticas.

Exemplos Práticos

Listar arquivos detalhados no diretório home:

ls -lah /home/usuario

Buscar palavra "erro" em arquivos de log:

grep -i "erro" /var/log/syslog

Copiar diretório com todos arquivos e permissões:

cp -a /origem /destino

Alterar permissões para dono (leitura e escrita) e grupo (somente leitura):

chmod 640 arquivo.txt

Verificar uso de disco em human readable:

df -h

Conectar via SSH a um servidor remoto:

ssh usuario@192.168.1.10

Baixar arquivo da web:

wget https://exemplo.com/arquivo.zip

Dicas e Boas Práticas

  • Use sempre permissões mínimas necessárias para arquivos e diretórios.
  • Evite executar comandos com sudo sem necessidade para minimizar riscos.
  • Documente seus scripts e comandos para facilitar manutenção futura.
  • Use man comando para acessar a documentação local detalhada.
  • Mantenha o sistema atualizado e faça backups frequentes.

Tipos de Ataques e Pentest

Conhecer os tipos de ataques é essencial para proteger sistemas e realizar testes de invasão (pentest) de forma eficaz e ética.

Tipos Comuns de Ataques

  • Phishing: Ataques que enganam usuários para obter credenciais ou informações sensíveis.
  • SQL Injection: Inserção de código malicioso em formulários para acessar banco de dados indevidamente.
  • Cross-Site Scripting (XSS): Injeção de scripts maliciosos em páginas web para roubar dados ou controlar sessões.
  • Brute Force: Tentativa de descobrir senhas por força bruta, testando várias combinações.
  • Man-in-the-Middle (MitM): Interceptação de comunicação entre duas partes sem que elas percebam.
  • DoS/DDoS: Ataques para sobrecarregar servidores e indisponibilizar serviços.
  • Escalada de Privilégios: Exploração para obter níveis maiores de acesso no sistema.

Principais Ferramentas Kali Linux para Pentest

  • Nmap: Scanner de redes para descoberta de hosts e serviços. Exemplo: nmap -sS 192.168.1.0/24
  • Metasploit Framework: Plataforma para desenvolvimento e execução de exploits.
  • Aircrack-ng: Conjunto de ferramentas para auditoria de redes Wi-Fi.
  • Wireshark: Analisador de pacotes para captura e análise de tráfego de rede.
  • John the Ripper: Ferramenta para quebra de senhas usando ataques de dicionário e força bruta.
  • Burp Suite: Plataforma para testar segurança de aplicações web.
  • Hydra: Ferramenta de força bruta para serviços de autenticação.

Práticas de Pentest Ético

  • Obtenha sempre autorização escrita antes de realizar qualquer teste.
  • Documente todos os passos e resultados para análise posterior.
  • Utilize ambientes controlados (laboratórios) para treinamento e simulações.
  • Conheça as ferramentas antes de usar em ambientes produtivos.
  • Atualize-se constantemente sobre novas vulnerabilidades e técnicas de defesa.

Como Usar Ferramentas Kali Linux para Pentest

Nmap - Scanner de Rede

O nmap é usado para descobrir hosts ativos, portas abertas e serviços em uma rede.

Comando básico:

nmap -sS 192.168.1.0/24

Exemplo: Descobrir hosts ativos e portas TCP abertas na rede local.

Uso Ético: Sempre obtenha permissão antes de escanear redes que não são suas.

Hydra - Ataque de Força Bruta

O hydra permite realizar ataques de força bruta em serviços de autenticação como SSH, FTP e HTTP.

Comando básico:

hydra -l usuario -P lista_senhas.txt ssh://192.168.1.100

Explicação: Tenta logar no SSH do IP alvo usando o usuário usuario e uma lista de senhas do arquivo lista_senhas.txt.

Exemplo Prático: Usado em laboratório para testar robustez de senhas.

Dica: Nunca use em sistemas sem autorização explícita, é ilegal.

Metasploit Framework

O msfconsole é uma plataforma poderosa para criar e executar exploits contra alvos vulneráveis.

Passos básicos:

  1. Inicie o Metasploit: msfconsole
  2. Use um módulo exploit: use exploit/windows/smb/ms17_010_eternalblue
  3. Configure o alvo: set RHOSTS 192.168.1.105
  4. Execute: exploit

Nota: Use sempre em ambientes controlados para aprendizado ou com permissão.

Aircrack-ng - Teste de Segurança Wi-Fi

O conjunto aircrack-ng é utilizado para capturar e analisar tráfego Wi-Fi e tentar quebrar chaves WEP/WPA.

Passos básicos:

  • Coloque a placa wireless em modo monitor: airmon-ng start wlan0
  • Capture pacotes: airodump-ng wlan0mon
  • Capture handshake de uma rede alvo
  • Quebre a senha com o dicionário: aircrack-ng -w wordlist.txt capture.cap

Aviso: Use somente em redes autorizadas.

Wireshark - Análise de Pacotes

Wireshark captura e analisa tráfego de rede em tempo real, ótimo para diagnóstico e investigação.

Uso: Inicie a captura na interface desejada e use filtros para focar em pacotes específicos.

Dicas para Pentest Seguro e Ético

  • Sempre tenha autorização escrita antes de realizar qualquer teste.
  • Realize testes em ambientes controlados, como máquinas virtuais ou laboratórios isolados.
  • Documente tudo: passos, comandos, resultados e recomendações.
  • Respeite privacidade e confidencialidade dos dados.
  • Esteja sempre atualizado sobre leis e regulamentações locais.
