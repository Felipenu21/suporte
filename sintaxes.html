Sintaxes CMD

O prompt de comando do Windows (CMD) utiliza comandos simples, que aceitam parâmetros para executar tarefas no sistema. A sintaxe geral é:

comando [parâmetros] [opções]

Exemplo de comando simples:

dir /p /w

Este comando lista os arquivos e pastas do diretório atual, pausando página por página e mostrando em colunas.

Sintaxes PowerShell

O PowerShell é uma shell mais poderosa, orientada a objetos, que aceita cmdlets, scripts e funções. A sintaxe básica é:

Cmdlet-Name [-Parâmetro Valor] [-Flag]

Exemplo:

Get-Process -Name "notepad"

Sintaxes Linux

No Linux, o terminal usa comandos típicos do Unix/Linux. A sintaxe geral é:

comando [opções] [argumentos]

Exemplo:

ls -l /home/usuario

Flags e Parâmetros Comuns

Flags são opções que modificam o comportamento dos comandos, geralmente precedidas por `-` (Linux, PowerShell) ou `/` (CMD).

  • -h ou --help: exibe ajuda
  • -v ou --verbose: mostra detalhes da execução
  • -q ou --quiet: executa silenciosamente
  • /s (CMD): aplica recursão
  • /q (CMD): modo silencioso
  • -Force (PowerShell): força a execução, mesmo com avisos

Exemplos Práticos

Listar arquivos no CMD com paginação:

dir /p

Obter lista de processos no PowerShell:

Get-Process | Where-Object {$_.CPU -gt 100}

Copiar arquivos no Linux, mostrando progresso:

rsync -av --progress /origem/ /destino/

Remover diretório e todo conteúdo no CMD:

rmdir /s /q C:\PastaTeste

Alterar permissões no Linux:

chmod 755 script.sh
