Introdução ao Nmap

O Nmap (Network Mapper) é uma ferramenta open source utilizada para exploração e auditoria de segurança em redes de computadores.

Ele é amplamente usado para descobrir hosts e serviços em uma rede, além de mapear portas abertas e identificar vulnerabilidades.

Instalação

Você pode instalar o Nmap em diferentes sistemas operacionais:

  • Windows: Baixe o instalador oficial em https://nmap.org/download.html.
  • Linux: Utilize o gerenciador de pacotes da sua distribuição, por exemplo:
    sudo apt install nmap (Debian/Ubuntu)
    sudo yum install nmap (CentOS/Fedora)
  • MacOS: Instale via Homebrew:
    brew install nmap

Comandos Básicos

Veja alguns comandos básicos para iniciar:

ComandoDescrição
nmap 192.168.1.1Escaneia o host 192.168.1.1 para portas comuns.
nmap -sP 192.168.1.0/24Ping scan para descobrir hosts ativos na rede.
nmap -p 80,443 192.168.1.1Escaneia as portas 80 e 443 do host.
nmap -v 192.168.1.1Escaneia com modo verbose (detalhado).
nmap -sS 192.168.1.1Escaneamento SYN stealth.

Comandos Avançados

Comandos para uso avançado e auditoria detalhada:

  • -O: Detecta o sistema operacional do host.
  • -A: Ativa detecção de sistema operacional, versão, script scanning e traceroute.
  • --script vuln: Executa scripts para detectar vulnerabilidades conhecidas.
  • -p-: Escaneia todas as 65535 portas TCP.
  • -sU: Escaneia portas UDP.
  • --open: Exibe apenas portas abertas.
  • -Pn: Não faz ping antes do escaneamento (útil para hosts que bloqueiam ping).

Exemplos Práticos

Escanear rede local inteira:

nmap -sP 192.168.1.0/24

Escanear portas específicas:

nmap -p 22,80,443 192.168.1.10

Detectar sistema operacional e serviços:

nmap -A 192.168.1.10

Escanear todas as portas TCP:

nmap -p- 192.168.1.10

Executar script de vulnerabilidades:

nmap --script vuln 192.168.1.10

Dicas e Boas Práticas

  • Use o Nmap sempre com permissão para evitar problemas legais.
  • Combine opções para obter resultados mais detalhados, mas cuide do tempo de escaneamento.
  • O Nmap pode ser usado em conjunto com outras ferramentas de segurança.
  • Leia a documentação oficial para aproveitar scripts e opções avançadas: Documentação Nmap.
  • Use a opção -oN para salvar o resultado em arquivo de texto.
