O que é o Executar

O recurso Executar do Windows permite executar comandos, abrir programas, pastas, documentos ou páginas da web rapidamente, digitando o nome ou caminho desejado.

Ele é muito útil para acesso rápido a configurações e ferramentas do sistema.

Como Abrir o Executar

  • Pressione Win + R no teclado.
  • Ou clique no menu iniciar, digite "Executar" e pressione Enter.

Comandos Comuns no Executar

Aqui está uma lista dos comandos mais usados, com descrições:

cmdAbre o Prompt de ComandoJanela do terminal CMD do Windows.
powershellAbre o PowerShellJanela do PowerShell para comandos avançados.
explorerAbre o Explorador de ArquivosAbre uma janela do gerenciador de arquivos.

services.mscAbre o Gerenciador de ServiçosPara iniciar, parar e configurar serviços.
msconfigAbre Configuração do SistemaConfigura inicialização e serviços.
regeditAbre o Editor do RegistroPara editar o registro do Windows (cuidado!).
taskmgrAbre o Gerenciador de TarefasGerenciar processos e performance.
notepadAbre o Bloco de NotasEditor de texto simples.
mspaintAbre o PaintEditor de imagens básico.

Exemplos Práticos

Executar o Prompt de Comando como administrador:

  1. Pressione Win + R.
  2. Digite cmd e pressione Ctrl + Shift + Enter para abrir como administrador.

Abrir uma pasta específica:

explorer C:\Users\SeuUsuario\Documents

Abrir o Editor do Registro:

regedit

Dicas Extras

  • Você pode usar variáveis de ambiente no Executar, como %appdata%, %temp%, %systemroot%.
  • Use o botão de histórico no Executar para acessar comandos usados anteriormente.
  • É possível criar atalhos para comandos personalizados, facilitando o uso.
