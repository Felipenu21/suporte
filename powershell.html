Comandos Básicos do PowerShell

PowerShell é uma poderosa shell e linguagem de scripts da Microsoft usada para automação e gerenciamento do sistema. Começaremos com os comandos básicos que todo iniciante deve conhecer.

Comando Descrição Exemplo
Get-ChildItem (ou ls) Lista arquivos e pastas no diretório atual ou especificado. 
Get-ChildItem
ls C:\Windows
Set-Location (ou cd) Muda o diretório atual. 
Set-Location C:\Users
cd ..
cd "C:\Program Files"
Get-Help Exibe ajuda sobre um comando. 
Get-Help Get-ChildItem
Get-Help Get-ChildItem -Full
Get-Content Mostra o conteúdo de arquivos texto. 
Get-Content .\arquivo.txt
Copy-Item Copia arquivos ou pastas. 
Copy-Item .\arquivo.txt D:\Backup\arquivo.txt
Move-Item Move arquivos ou pastas. 
Move-Item .\arquivo.txt D:\Backup\
Remove-Item Remove arquivos ou pastas. 
Remove-Item .\arquivo.txt

Comandos Avançados do PowerShell

Comandos avançados para administração, filtragem e automação.

Comando Descrição Exemplo
Where-Object Filtra objetos com base em uma condição. 
Get-Process | Where-Object { $_.CPU -gt 100 }
Sort-Object Ordena objetos por propriedade. 
Get-Service | Sort-Object Status
Select-Object Seleciona propriedades específicas para exibir. 
Get-Process | Select-Object Name, Id, CPU
Invoke-Command Executa comandos/scripts remotamente. 
Invoke-Command -ComputerName Server01 -ScriptBlock { Get-Service }
Start-Service, Stop-Service, Restart-Service Gerencia serviços do Windows. 
Start-Service spooler
Stop-Service wuauserv
Restart-Service bits

Scripts PowerShell

Automatize tarefas com scripts (.ps1). Veja exemplos e estrutura básica:

Exemplo básico

# Lista processos com CPU > 50
$processos = Get-Process | Where-Object { $_.CPU -gt 50 }
foreach ($proc in $processos) {
  Write-Output "Processo: $($proc.Name) - CPU: $($proc.CPU)"
}

Executar script

Execute via PowerShell:

.\script.ps1

Se houver erro, ajuste política de execução:

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Parâmetros em scripts

param (
  [string]$Nome = "Usuário",
  [int]$Idade = 18
)

Write-Output "Nome: $Nome"
Write-Output "Idade: $Idade"

Sintaxes e Flags Importantes

Parâmetros comuns que modificam o comportamento dos comandos.

Flag Descrição
-RecurseProcessa itens recursivamente (subpastas).
-ForceForça a execução mesmo com restrições.
-WhatIfSimula o comando sem executar (teste).
-ConfirmPede confirmação antes de executar.
-ErrorActionDefine ação em caso de erro (ex: SilentlyContinue).
-VerboseMostra detalhes da execução.

Exemplo de uso com flags

Remove-Item C:\Temp\* -Recurse -Force -WhatIf

Simula remoção de tudo em C:\Temp, incluindo subpastas, forçando a ação, mas sem excluir realmente.

Exemplos Práticos

  • Listar usuários locais: Get-LocalUser
  • Ver espaço em disco: Get-PSDrive -PSProvider FileSystem
  • Parar processo pelo nome: Stop-Process -Name "notepad"
  • Exportar processos para CSV: Get-Process | Export-Csv -Path processos.csv -NoTypeInformation
  • Agendar tarefa diária simples: Register-ScheduledTask -TaskName "BackupDiario" -Trigger (New-ScheduledTaskTrigger -Daily -At 3am) -Action (New-ScheduledTaskAction -Execute "powershell.exe" -Argument "-File C:\Scripts\backup.ps1")

Scripts Úteis para Suporte Técnico

Scripts que facilitam tarefas comuns de suporte técnico e administração de máquinas Windows.

1. Backup simples de uma pasta

# Faz backup da pasta Documentos para D:\Backup\Documentos
$origem = "C:\Users\$env:USERNAME\Documents"
$destino = "D:\Backup\DocumentsBackup_$(Get-Date -Format yyyyMMddHHmmss)"
Copy-Item -Path $origem -Destination $destino -Recurse -Force
Write-Output "Backup concluído em $destino"

2. Verificar status de serviços essenciais

$servicos = @("wuauserv", "Spooler", "bits")
foreach ($servico in $servicos) {
  $status = Get-Service -Name $servico
  Write-Output "$($status.DisplayName): $($status.Status)"
}

3. Limpar arquivos temporários do usuário

$temp = "$env:LOCALAPPDATA\Temp\*"
Remove-Item -Path $temp -Recurse -Force -ErrorAction SilentlyContinue
Write-Output "Arquivos temporários removidos."

4. Monitorar uso de CPU e memória em tempo real

while ($true) {
  $cpu = (Get-Counter '\Processor(_Total)\% Processor Time').CounterSamples[0].CookedValue
  $mem = (Get-Counter '\Memory\Available MBytes').CounterSamples[0].CookedValue
  Clear-Host
  Write-Output ("Uso CPU: {0:N2}%" -f $cpu)
  Write-Output ("Memória disponível: {0:N0} MB" -f $mem)
  Start-Sleep -Seconds 3
}

5. Reiniciar um serviço específico

$servico = "Spooler"
Restart-Service -Name $servico -Force
Write-Output "Serviço $servico reiniciado."

Conclusão

PowerShell é uma ferramenta essencial para suporte e administração Windows, combinando comandos básicos e avançados, scripts flexíveis e flags úteis para tarefas do dia a dia e automação avançada.
