Introdução ao PowerShell

PowerShell é uma poderosa shell e linguagem de scripts desenvolvida pela Microsoft, amplamente utilizada para automação, gerenciamento e administração de sistemas Windows e também multiplataforma (Linux e Mac).

Este tutorial vai guiá-lo desde conceitos básicos até comandos e scripts avançados para suportar seu dia a dia técnico.

Comandos Básicos do PowerShell

Comandos fundamentais para navegação, visualização e manipulação de arquivos e processos.

Comando Descrição Exemplo
Get-ChildItem (ou ls) Lista arquivos e pastas no diretório atual ou especificado. 
Get-ChildItem
ls C:\Windows
Set-Location (ou cd) Muda o diretório atual. 
Set-Location C:\Users
cd ..
Get-Help Exibe ajuda detalhada de comandos. 
Get-Help Get-ChildItem -Full
Get-Content Mostra o conteúdo de arquivos texto. 
Get-Content .\arquivo.txt
Copy-Item Copia arquivos ou pastas. 
Copy-Item .\arquivo.txt D:\Backup\arquivo.txt
Move-Item Move arquivos ou pastas. 
Move-Item .\arquivo.txt D:\Backup\
Remove-Item Remove arquivos ou pastas. 
Remove-Item .\arquivo.txt

Variáveis

Armazenam dados para reutilização no script ou linha de comando.

Exemplo de declaração e uso:

$nome = "Felipe"
$idade = 30
Write-Output "Nome: $nome, Idade: $idade"

Controle de Fluxo

Condicionais, loops e controle lógico.

If / Else

$idade = 20
if ($idade -ge 18) {
  Write-Output "Maior de idade"
} else {
  Write-Output "Menor de idade"
}

Loops

ForEach e While são os mais comuns:

# ForEach
$nomes = @("Ana", "Bruno", "Carlos")
foreach ($nome in $nomes) {
  Write-Output "Nome: $nome"
}

# While
$i = 0
while ($i -lt 3) {
  Write-Output "Número: $i"
  $i++
}

Funções

Reutilize código criando funções:

function Saudacao($nome) {
  Write-Output "Olá, $nome!"
}

Saudacao "Felipe"
Saudacao "Maria"

Módulos e Pacotes

PowerShell permite estender funcionalidades com módulos:

  • Get-Module: lista módulos carregados
  • Import-Module: importa módulo
  • Install-Module: instala módulo da galeria PowerShell
Get-Module -ListAvailable
Import-Module -Name Pester
Install-Module -Name Az -Scope CurrentUser

Tratamento de Erros

Use try, catch e finally para controlar erros:

try {
  Remove-Item -Path "C:\inexistente.txt" -ErrorAction Stop
} catch {
  Write-Output "Arquivo não encontrado: $_"
} finally {
  Write-Output "Finalizado"
}

Scripts PowerShell

Automatize tarefas criando arquivos .ps1.

Exemplo básico

# Lista processos com CPU > 50
$processos = Get-Process | Where-Object { $_.CPU -gt 50 }
foreach ($proc in $processos) {
  Write-Output "Processo: $($proc.Name) - CPU: $($proc.CPU)"
}

Salvar saída em arquivo

Get-Process | Out-File C:\Temp\processos.txt

Sintaxes e Flags Comuns

Parâmetros úteis para controlar execução dos comandos:

  • -Recurse: inclui subdiretórios
  • -Force: força ação mesmo em arquivos ocultos/somente leitura
  • -WhatIf: simula ação sem executá-la
  • -Confirm: pede confirmação antes da ação
  • -ErrorAction SilentlyContinue: oculta mensagens de erro
  • -Verbose: mostra informações detalhadas da execução

Exemplos Práticos

Algumas tarefas comuns no suporte técnico:

Backup simples de arquivos

Copy-Item C:\Dados\* D:\Backup\ -Recurse -Force

Limpar arquivos temporários

Remove-Item "C:\Windows\Temp\*" -Recurse -Force

Monitorar processos que usam muita CPU

Get-Process | Where-Object { $_.CPU -gt 100 } | Sort-Object CPU -Descending

Parar um serviço

Stop-Service -Name "wuauserv"

Iniciar um serviço

Start-Service -Name "spooler"

Scripts Úteis para Suporte Técnico

Automatize tarefas frequentes para agilizar o atendimento:

  • Verificar serviços essenciais rodando:
    • Get-Service -Name "wuauserv", "bits", "Spooler" | Where-Object {$_.Status -ne "Running"}
  • Enviar logs por email (requer SMTP configurado):
    • $smtpServer = "smtp.exemplo.com"
$smtpFrom = "suporte@empresa.com"
$smtpTo = "admin@empresa.com"
$messageSubject = "Log do Sistema"
$messageBody = Get-Content -Path "C:\Logs\log.txt" | Out-String
Send-MailMessage -From $smtpFrom -To $smtpTo -Subject $messageSubject -Body $messageBody -SmtpServer $smtpServer
  • Agendar tarefa automatizada:
    • $action = New-ScheduledTaskAction -Execute 'PowerShell.exe' -Argument '-File C:\Scripts\backup.ps1'
$trigger = New-ScheduledTaskTrigger -Daily -At 22:00
Register-ScheduledTask -TaskName "Backup Diário" -Action $action -Trigger $trigger -Description "Backup automático diário às 22h"
