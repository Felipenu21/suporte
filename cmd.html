Comandos Básicos do CMD

O Prompt de Comando (CMD) é uma interface de linha de comando presente no Windows, essencial para suporte técnico, scripts simples e diagnósticos rápidos.

Comando Descrição Exemplo
dir Lista arquivos e pastas do diretório atual. 
dir
dir C:\Windows
cd Altera o diretório atual. 
cd ..
cd C:\Users
copy Copia arquivos. 
copy arquivo.txt D:\Backup\
move Move arquivos. 
move arquivo.txt D:\Backup\
del Deleta arquivos. 
del arquivo.txt
mkdir Cria diretório. 
mkdir NovaPasta
rmdir Remove diretório vazio. 
rmdir PastaVelha
cls Limpa a tela do prompt. 
cls
ipconfig Exibe configurações IP da máquina. 
ipconfig /all
ping Testa conexão com outro host. 
ping google.com

Comandos Avançados do CMD

Comandos avançados para diagnóstico, gerenciamento de rede e scripts mais complexos.

Comando Descrição Exemplo
tasklist Lista processos em execução. 
tasklist
taskkill Encerra processo pelo nome ou PID. 
taskkill /IM notepad.exe /F
taskkill /PID 1234 /F
netstat Mostra conexões de rede e portas usadas. 
netstat -an
systeminfo Exibe informações do sistema. 
systeminfo
chkdsk Verifica e repara erros no disco. 
chkdsk C: /F /R
sfc Verifica e repara arquivos do sistema. 
sfc /scannow
shutdown Desliga ou reinicia o computador. 
shutdown /r /t 0
net user Gerencia usuários locais. 
net user
net user username /add
diskpart Gerenciamento avançado de discos. Executado interativamente:
diskpart

Flags e Parâmetros Importantes

Algumas flags comuns em comandos do CMD para modificar comportamentos.

Flag Descrição Exemplo
/F Força a execução (ex: forçar encerramento do processo). 
taskkill /IM notepad.exe /F
/S Especifica computador remoto. 
shutdown /s /t 0 /m \\PC-Remoto
/T Define tempo para execução (em segundos). 
shutdown /r /t 30
/Q Modo silencioso (ex: desligar sem aviso). 
shutdown /s /t 0 /f /q
/A Aborta desligamento em andamento. 
shutdown /a

Exemplos Práticos

  • Limpar cache DNS: ipconfig /flushdns
  • Ver conexões ativas: netstat -an
  • Mostrar configuração IP: ipconfig /all
  • Fechar o bloco de notas: taskkill /IM notepad.exe /F
  • Desligar o PC em 60 segundos: shutdown /s /t 60
  • Criar usuário local: net user usuario123 senha123 /add
  • Ver tarefas agendadas: schtasks /query

Scripts e Dicas para Suporte Técnico

Automatize tarefas comuns e resolva problemas com estes scripts simples em batch (.bat) para suporte.

1. Script para limpar arquivos temporários do Windows

@echo off
echo Limpando arquivos temporários...
del /s /q %temp%\*
del /s /q C:\Windows\Temp\*
echo Limpeza concluída.
pause

2. Script para reiniciar serviços essenciais

@echo off
echo Reiniciando serviços essenciais...
net stop wuauserv
net start wuauserv
net stop spooler
net start spooler
net stop bits
net start bits
echo Serviços reiniciados.
pause

3. Script para verificar conectividade e salvar resultado

@echo off
echo Testando conectividade com google.com...
ping google.com > "%userprofile%\desktop\ping_log.txt"
echo Resultado salvo em ping_log.txt na área de trabalho.
pause

4. Script para backup simples de uma pasta

@echo off
set origem=C:\Users\%username%\Documents
set destino=D:\Backup\DocumentsBackup_%date:~6,4%%date:~3,2%%date:~0,2%
echo Fazendo backup da pasta Documentos...
xcopy "%origem%" "%destino%" /E /I /H /C /Y
echo Backup concluído em %destino%
pause

5. Script para liberar cache DNS

@echo off
ipconfig /flushdns
echo Cache DNS liberado.
pause

Conclusão

O CMD é uma ferramenta poderosa para suporte técnico e administração de sistemas Windows, que com prática permite automatizar tarefas, diagnosticar problemas e controlar sistemas remotamente.
