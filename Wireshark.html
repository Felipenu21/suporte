Introdução ao Wireshark

Wireshark é um analisador de protocolo de rede amplamente utilizado para captura e análise de pacotes em redes de computadores. É uma ferramenta essencial para diagnóstico, segurança e análise de tráfego.

Instalação

Para instalar o Wireshark em sistemas Linux:

sudo apt update
sudo apt install wireshark

Em Windows e macOS, baixe o instalador oficial em wireshark.org.

Captura de Pacotes

Para iniciar uma captura, selecione a interface de rede correta. Exemplo no terminal:

sudo wireshark

Ou para capturar diretamente no terminal e salvar:

sudo tshark -i eth0 -w captura.pcap

Filtros

Filtros são essenciais para focar na análise dos dados relevantes.

Filtros de captura

  • tcp port 80 — captura somente pacotes TCP na porta 80
  • host 192.168.1.10 — captura pacotes de/para o IP especificado

Filtros de exibição

  • http — exibe apenas pacotes HTTP
  • ip.addr == 192.168.1.10 — exibe pacotes com esse IP origem ou destino
  • tcp.flags.syn == 1 and tcp.flags.ack == 0 — pacotes SYN (início de conexão TCP)

Análise de Pacotes

Wireshark permite examinar o conteúdo dos pacotes, identificar protocolos, endereços, flags e dados transmitidos. É possível inspecionar pacotes individuais e seguir fluxos de comunicação.

Exemplo Prático

Capturando pacotes HTTP para analisar uma requisição web:

sudo wireshark

Abra a interface de rede, comece a captura, faça uma navegação web, e use o filtro http para visualizar as requisições.

Dicas Importantes

  • Use Wireshark com privilégios adequados (sudo/root no Linux) para capturar tráfego.
  • Esteja ciente das implicações legais e éticas ao capturar tráfego de redes alheias.
  • Utilize filtros para reduzir o volume e focar na análise.
  • Salve capturas para análises futuras com formato .pcap.

Comandos Úteis do Tshark (versão terminal do Wireshark)

sudo tshark -D                  # Listar interfaces disponíveis
sudo tshark -i eth0 -c 100       # Capturar 100 pacotes na interface eth0
sudo tshark -r captura.pcap      # Ler arquivo de captura
sudo tshark -Y "http.request"    # Filtrar pacotes HTTP na leitura
